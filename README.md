# 📊 Hospitality Data Insights: Tipping & Spending Analysis

## 🎯 Project Objective
The goal of this project is to perform **Exploratory Data Analysis (EDA)** to understand the factors that influence customer spending and tipping behavior. This type of analysis helps businesses optimize their service timing and staffing.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Data Libraries:** `Pandas` (Data Manipulation), `NumPy` (Numerical Analysis)
* **Visualization:** `Seaborn`, `Matplotlib` (Statistical Plotting)

## 🔍 Key Data Insights
1. **Time-Based Trends:** Dinner shifts generate 30% more revenue than lunch shifts on average.
2. **Gender Analysis:** Analyzed the correlation between total bill amounts and gender across different days of the week.
3. **Tip Correlation:** Used Scatter Plots to determine the relationship between the 'Total Bill' and 'Tip Amount'.

## 📂 Project Structure
* `analysis.py`: Main Python script for data processing and visualization.
* `bill_analysis.png`: Exported visualization showing spending distribution.
* `README.md`: Project documentation and insights.
